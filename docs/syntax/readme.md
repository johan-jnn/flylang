# The language's syntax

Here you'll learn on how to write flylang code.
Note that at this time the syntax may change.

To work with the flylang programming language's syntax, you need to have in mind 3 type of things :

- [The instructions](./instructions/readme.md)
  - To make statements
- [The expressions](./expressions/readme.md)
  - To reference values
- [The definables](./definables/readme.md) (Which are a specific type of expressions)
  - To define values
